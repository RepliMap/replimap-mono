# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities to **hello@replimap.com**.

**Do NOT open public GitHub issues for security bugs.**

We will respond within 48 hours and work with you to resolve the issue.

## Scope

This repository contains only the **frontend UI code**.

The core RepliMap CLI:
- Runs 100% locally on your machine
- Does not transmit infrastructure data to any server
- Source code is in a separate repository

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
