# Welcome to RepliMap

RepliMap is an AWS Infrastructure Intelligence Engine that reverse-engineers your AWS infrastructure into Terraform code.

## What is RepliMap?

RepliMap scans your AWS account and:

- **Generates Terraform** - Production-ready IaC from existing resources
- **Detects Drift** - Find differences between state and reality
- **Maps Dependencies** - Visualize resource relationships
- **Creates IAM Policies** - Least-privilege policies based on actual usage

## Key Features

- 100% Local Execution - Your credentials never leave your machine
- Lightning Fast - Scan thousands of resources in seconds
- 50+ AWS Services - VPC, EC2, RDS, Lambda, S3, IAM, and more
